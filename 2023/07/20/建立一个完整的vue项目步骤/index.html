vue脚手架项目建立

  • 基础：下载vscode，配置好环境变量；在终端命令导入node和npm包管理工具，配置好系统环境变量，并且注意版本适配问题，由于npm是国外请求，最好配置国内cnpm淘宝镜像
  • 第一步：npm install -g @vue/cli（安装全局的vue脚手架）
  • 第二步: vue create vue-router(创建一个vue项目根文件)
  • 第三步：npm run serve(运行)

mian.js

// 创建 vue 实例

const app = createApp(App);

// 挂载vue 实例

app.mount(‘#app’)

导入echarts

  • 第一步：npm install echarts –save （npm安装ECharts）

  • 第二步：

    // 引入 echarts

    import * as echarts from ‘echarts’

    // 全局挂载 echarts

    app.config.globalProperties.$echarts = echarts

    • eslint是语法检查工具，缺点是对所写代码要求过于严格；可以在package-json中rule（规则）加人”no-mixed-spaces-and-tabs”:0，可以减少因为不必要的小格式报错出错

  • 第三步：引用运行即可

element-plus

  • 安装：npm install element-plus --save
  • 安装图标：npm install @element-plus/icons-vue
  • // 导入 element-plus import ElementPlus from "element-plus"; // 导入 element-plus 样式 import "element-plus/dist/index.css"; // 导入 element-plus 图标 import * as ElementPlusIconsVue from "@element-plus/icons-vue"; //在app.mount前全局挂载 app.use(ElementPlus) // 注册所有图标 for (const [key, component] of Object.entries(ElementPlusIconsVue)) { app.component(key, component); }

导入axios

  • 第一步: npm install –save axios vue-axios 下载包
  • 第二步：在main.js文件中对其进行导入和注册
    // 导入vue-axios模块
    import VueAxios from ‘vue-axios’
    import axios from ‘axios’;
    // 注册axios
    app.use(VueAxios, axios)

导入router

  • 第一步:npm install vue-router@4 -s 下载依赖

一、简介： Vue路由是用来管理页面切换或跳转的一种方式。

二、安装：npm install vue-router@4 -s

三、在component下新增两个组件：

 编写简单内容用标识

页面1
页面2

 router-view：用于渲染

五、在main.js中编写代码
1.导入路由
import {createRouter,createWebHashHistory} from ‘vue-router’;

2.导入组件
import Demo1 from './components/Demo1.vue';
import Demo2 from './components/Demo2.vue';

3.映射路径
const routes = [
  {path:'/demo1',component:Demo1},
  {path:'/demo2',component:Demo2},
]

4.路径映射绑定
const router = createRouter({
  history:createWebHashHistory(),
  routes:routes
}) 

5.使用：
app.use(router);
