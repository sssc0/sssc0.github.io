#HTML

div盒子与标签类型

1、div标签及使用（基本属性）

给div定义外边距时，不会改变盒子大小；但是定义内边距时，会改变盒子大小样式的。

定义水槽，尽可能使用外边距定义

2、标签类型（基本标签特性）

overflow 用于容器里面内容超过容器本身高度及宽度如何显示超出的部分

（1）块级元素

​ 设置display:block就是将元素显示为块级元素

​ 独占一行，元素的高宽顶底边距都可设置，并且元素宽度在不设置的情况下，是他本身父容器的> 100%（和父元素的宽度一致）

​ 例:div、p、h1、ol、ul、dl、table、form

（2）内联（行内）元素

​ 可以通过display:inline将元素设置为内联元素

​ 和其他元素在同一行上。（不会独占一行）

​ 元素的高度、宽度、以及顶部和底外边距都不可设置

​ 元素的宽度就是它所包含的文本或者图片的宽度，且不可通过属性改变

​ 例：a、span、br 、i、em、strong、label、q

（3）内联（行内）块级元素

​ 可以通过display:inline-block将元素设置为内联块级元素。

​ 和其他元素在同一行上。（不会独占一行）

​ 元素的高度、宽度、以及顶和底边距都可设置

​ 例：img、input

3、标签布局、排版

​ （1）flow 流动模型

​ （2）float 浮动模型

​ （3）position 层模型

​ a、relative：相对布局 会在文本流中留下一个位置并且隐藏，并且其他流只会承认原来位置的流

​ b、absolute：绝对布局 不会留下位置 两个布局都会跟着父类动（父类必须是非静态父类）

​ c、static：静态布局

​ d、fixed：黏着布局 例：页面主导航、头部导航固定显示。

4、常用的自适应布局（常用的布局方式）

​ （1）主div大小固定，且伸缩居中，子div浮动布局，水槽固定

3.属性选择器

语法:E[attr 选择规则 “value”]

E表示选择器节点，attr表示html元素内的属性名，value表示html属性的属性值

例p[class=”p1”]

(1)E[attr] —-属性匹配（只匹配属性，不在乎值）

(2)E[attr=”value”] —–全等匹配

(3)E[attr~=”value”] ——列表包含匹配

(4)E[attr^=”value”] ——头部匹配

​ 只对参数列表里的第一个参数值有用

（5）E[attr$=”value”] ——尾部匹配

​ 只对参数列表里的最后一个参数值有用

（6）E[attr*=”value”] ——-字符包含匹配

JS

js

1、js是一门弱类型、轻量级的编程语言。但严格区分大小写

2、js会自动加分号；但是解析器要花费时间。

3、js中的代码块》》》用花括号括起来的代码是用来提高代码可读性。js中没有作用域

4、js中一个函数，有一个生命周期，或一个代码块叫做一个作用域，里面定义的变量不能在外部使用。在作用域外，或者其函数退出，里面的变量就会被销毁。

5、js有五种数据类型：undefined（表示未赋值）、null、boolean、number、string

当输入的数据类型不匹配时，js会自动强转数据类型。（java中需要手动强转）

6、js中布尔类型转换》》》string任何非空字符强转为true，空字符强转为flase。

​ number任何非零数值转化为true，0和NaN转化为flase。

​ object任何对象转化为true，null转化为flase。

​ undefined 中not applicable（不适用）转化为true，undefined转化为false

tostring（2）方法可以将其他类型以2进制转化为字符串(注：这个方法只是转化为字符串并且返回，而不是直接转化)

var a = 16；

var b = a.toString（）——–将a的值强制转换为字符型然后返回，用b接收返回值（返回值的类型转化了但a的类型还是原来不变）

var c = Number（a） ——–将a的值强制转化为数值型然后返回，用c接收返回值

typeof：用于检查变量的数据类型。

例 var num = 16; console.log(typeof num.toString)

:转义字符：取消特殊语义 例： var a = (“ \ “你好 \ “”) console（a） 控制台输出 ”你好“ 将”转义成无意义字符

null与undefined不同，null有指针指向，而undefined没有指针指向

number 数据类型中的》》》NaN：表示一个状态，可以返回number，但是没有数值的状态

NaN的两个特点（1）任何涉及NaN的操作都会返回NaN本身。

​ （2）NaN不与任何值相等，包括NaN本身。

isNaN（）函数：判断传进来的参数（括号里的值）是否能被转换为number类型，如果能转换成number类型则不是NaN，返回的值为false。如果不能转换成number类型则是NaN，返回的值为true。

运算符

a++：在表达式执行后才执行++自增

++a：在表达式执行之前就++自增

++比+的运算优先级高

==相等于： 在比较之前会执行一次强转，将两个比较值转换成同类型。

===全等于：在比较之前不会强转，会比较两个比较值的数据类型。

短路与或 当前面的条件已经足以影响整个逻辑结果时，短路后面的代码不会执行。

短路与》》 》 （a）&&（b） 如果a为false，则无论b为true还是false结果都为false。则b代码不再执行。

短路或》》》 （a）||（b） 如果a为true，则无论b为true还是false结果都为true。则b代码不再执行。
